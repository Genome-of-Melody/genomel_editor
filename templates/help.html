Annotating with Genomel Editor

To start annotating with the Genomel Editor, you need to:

  • Obtain a user account. (Contact Jan Hajiƒç at the Masaryk Institute and Archives, Czech Academy of Sciences.).
  • Log in to the editor using the form in the upper area site (header).
  • Go to the Annotate page.
  • The system will either assign to you a new annotation, or it will load a mellody for you to continue annotating.

Once you are on the annotation page, you have to 1) load the image, 2) syllabize the text, 3.1) transcribe the melody into Volpiano, or 3.2) mark the melody as impossible to transcribe (incomplete or adiastematic), and 4) save your progress.

1) Loading the image:

  • In order to load the image from which you are transcribing, you need to copy the image into the clipboard first.
  • Go to the image URL that the system provides you with. Some chant databases serve the image directly to your browser (like ispan.pl). In that case, by right-clicking the image in the browser, a context menu comes up that contains the "Copy Image" option. Click that option.
  • However, some online libraries, such as Gallica or the British National Library, don't allow you to copy the image directly. However, what they do provide is a download option. If you download the image, it opens in a new browser tab as a normal image file. You can then copy it from there using right-click to bring up the context menu and select "Copy Image".
  • Once the image is copied, switch back to the annotation tab and just press paste: Ctrl+V (or Cmd+V on Mac). The image will paste itself into the image area.
  • The image is draggable, and you can use the buttons to zoom in or out if need be. You can also adjust the size of the image area -- it is resizable using the bottom right corner.
  • Once you can comfortably see the part of the image that you should be transcribing, you can start working on the Volpiano transcription.

2) Syllabizing the text:

  • Under the image, you can see the "Syllabized text" field pre-populated with the manuscript text.
  • However, the information we donwloaded from Cantus Index does not contain the syllabization of the text. This has to be done manually (at least if you are the first person to be annotating a chant).
  • Add hyphens ("-") to indicate syllable breaks. Do not add any spacing.

3.1) Transcribing the melody:

  • Proceed to transcribe the melody into Volpiano according to the Cantus Volpiano Protocols (clicking the link opens a new tab).
  • At any point, to check your progress, click the "Render" button.
3.2) Marking the melody as impossible to transcribe:
  • It can happen that the chant comes from an adiastematic source. In that case, do not transcribe anything and check the "Adiastematic" checkbox.
  • In other cases, a chant might be recorded in an incomplete form. This most often happens when the chant occurs in the same source earlier, so possibly only the incipit is written down (so that the singers recall the melody, which they would have known by heart anyway). In this case, do not transcribe anything and check the "Incomplete" checkbox.

4) Save your work

  • In order to save your work in progress, click the "Save" button. The server should let you continue editing. (We retain the image in the browser's storage, so if you cleared your browsing data since last working on the melody, you will have to copy the image again.)
  • Once you are done with the melody transcription and text syllabization, make sure you render your transcription and check that the text and melody fit together properly. Then, to mark the transcription as complete, check the "Is transcribed?" checkbox. Then, click "Save". The melody will be moved to the next step of the process: validation.
